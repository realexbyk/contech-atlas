Skills · for Claude Cowork, Cursor & Codex

Teach your AI agent construction data.

Skills are packaged instructions, reference knowledge and scripts that turn a general AI agent into a construction-data specialist. Install them in Claude Cowork, drop them into a Cursor rules folder, or wire them into Codex workflows — same skill, three runtimes. Everything here maps to nodes on the Atlas.

Scheduling

XER Schedule Auditor

Parses Primavera P6 XER/XML files, runs DCMA-14-style quality checks (logic density, leads/lags, constraints, float), and produces a review memo with the offending activities listed.

IN .xer / .xml  →  OUT quality report + issue register

How skills fit the Atlas

Skills are the connective tissue.

Every skill on this page operates between nodes of the Atlas: the XER Auditor sits between Scheduling and Data Quality; the WBS Normalizer implements the Master Data governance node; the Data Product Scaffolder packages the Data Platform into owned Data Products. Agents don't replace the stack — they operate it.

Skills will ship as downloadable packs: a .skill bundle for Claude Cowork, a rules/commands folder for Cursor, and an agent workflow definition for Codex. Subscribe on the home page to hear when the first pack lands.
